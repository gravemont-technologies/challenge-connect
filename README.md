# Tahqeeq - Operational Excellence Platform

Tahqeeq is a Platform as a Service (PaaS) that connects businesses with GCC university students to handle operational tasks efficiently.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Tech Stack
- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
